## üöÄ Quick Start

1. **Install Dependencies**
```bash
pip install -r requirements.txt
```

2. **MongoDB Setup**
```bash
# Start MongoDB server
mongod

# Run the Flask API
python mongodb-flask-api.py
```

3. **LLM Setup**
```bash
# Start Ollama server
ollama serve

# Run the function calling demo
python ollama-function-call.py
```
